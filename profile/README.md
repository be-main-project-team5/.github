# 📖 프로젝트 소개

### 최애 아이돌의 스케줄, 더 이상 놓치지 마세요!

팬, 아이돌, 그리고 매니저가 함께 만들어가는 스케줄 관리 서비스, Idol-Schedule입니다. 팬들은 내가 좋아하는 아이돌과 그룹의 스케줄을 한눈에 모아보고, 아이돌과 매니저는 스케줄을 손쉽게 등록하고 관리할 수 있습니다.
